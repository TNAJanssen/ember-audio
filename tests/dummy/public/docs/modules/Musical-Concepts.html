Musical-Concepts

Provides helper classes that represent musical concepts meant to be used by classes from the Sound module.

This module has the following classes:

  • Beat
  • Note

This module has the following submodules:

  • Beat

    An instance of the Beat class represents a single "beat" on a rhythmic instrument.

  • Note

    The Note class provides identification-focused functionality for collections of musical notes.
