BeatTrack

Extends:

Sampler â†’ BeatTrack

An instance of this class has a single AudioBuffer (a single "sound") but provides methods to play that sound repeatedly and in a rhythmic way. An instance of this class behaves very similarly to a "lane" on a drum machine.

private _soundIterator: Iterator

Temporary storage for the iterable that comes from the _sounds Set. This iterable is meant to be replaced with a new copy every time it reaches it's end, resulting in an infinite stream of Sound instances.
