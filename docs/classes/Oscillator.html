Oscillator

Uses: Connectable Playable

A class that represents an oscillator for a synthesizer. Capable of creating and starting a waveform of specified type at a specified frequency.

All filters from BiquadFilterNode are available and can be enabled by providing a given filter's frequency.

// creates a sine wave oscillator with a highpass at 1000Hz
const audioContext = this.get('audio.audioContext');
const osc = Oscillator.create({
  audioContext,
  highpassFrequency: 1000
});
